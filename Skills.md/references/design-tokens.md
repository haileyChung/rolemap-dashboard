# 디자인 토큰 (Design Tokens) — 단일 출처

모든 색상·타이포·간격 값은 이 파일에 집중한다. 다른 Skills 파일에서 hex를 직접 쓰지 않고 토큰명을 참조하거나 본 파일을 인용한다. 데이터셋 교체와 무관하게 유지되는 시각 정체성 정의.

## 1. 색상 토큰 (Color)

### 1.1 의미 토큰 (Semantic)

| 토큰 | 값 | 의미 |
|---|---|---|
| `--bg` | `#0D1117` | 페이지 배경 (다크모드 베이스) |
| `--card` | `#161B22` | 카드/패널 배경 |
| `--border` | `#30363D` | 테두리 |
| `--text` | `#E6EDF3` | 본문 텍스트 |
| `--mute` | `#8B949E` | 보조 텍스트·라벨 |

### 1.2 시그널 토큰 (Signal — 판단·강조)

| 토큰 | 값 | 의미 | 사용처 예시 |
|---|---|---|---|
| `--green` | `#1D9E75` | 양수·우수·BUY | 수익률 상승, ROE > 15%, BUY 배지 |
| `--red` | `#E24B4A` | 음수·위험·SELL | 수익률 하락, MDD < -15%, SELL 배지 |
| `--gold` | `#C8963E` | 주의·강조 | 부채비율 200~400%, 1위 종목 30%↑ |
| `--teal` | `#3A9E9E` | 포인트·기본 액센트 | ETF 라인, 활성 토글, 패시브 뱃지 |
| `--gray` | `#888780` | 중립·벤치마크 | 벤치마크 라인(점선), HOLD 배지 |

### 1.3 차트 보조

- ETF 라인: `--teal` (`#3A9E9E`), 두께 2px, 실선
- 벤치마크 라인: `--gray` (`#888780`), 두께 1px, `borderDash: [5,5]`
- 도넛 팔레트: `--teal` 기준 명도 7단계
  ```
  ['#3A9E9E','#4DAEAE','#5FB8B8','#80C4C4','#9FCFCF','#BCDBDB','#D8E7E7','#888780']
  ```
- 골드 그라디언트 fill: `rgba(200,150,62,0.12)`
- 호버 행 배경: `rgba(255,255,255,0.02)`

### 1.4 조건부 색상 매핑 (`2-skills-criteria.md` 임계값 연동)

| 지표 | 임계값 | 토큰 | 적용 |
|---|---|---|---|
| MDD | < -15% | `--red` | KPI 카드 테두리 + 배너 |
| MDD | -15% ~ -5% | `--gold` | KPI 카드 테두리 |
| Sharpe | > 1.0 | `--green` | KPI 값 |
| Sharpe | < 0 | `--red` | KPI 값 |
| ROE | > 15% | `--green` | KPI 값 |
| ROE | < 5% | `--red` | KPI 값 |
| 부채비율 | > 400% (비금융) | `--red` | 진행바 |
| 부채비율 | > 200% (비금융) | `--gold` | 진행바 |
| 트래킹 에러 | < 2% | `--green` | KPI 값 |
| 트래킹 에러 | > 4% | `--gold` | KPI 값 |
| Beta | > 1.2 | `--gold` | 리스크 셀 |
| Beta | < 0.8 | `--green` | 리스크 셀 |
| VaR 95% | < -5% | `--red` | 리스크 셀 |
| VaR 95% | < -3% | `--gold` | 리스크 셀 |
| 1위 종목 비중 | > 30% | `--gold` | 테이블 행 |
| PER | < 섹터평균 × 0.8 | `--green` | 테이블 셀 |
| PER | > 섹터평균 × 1.2 | `--red` | 테이블 셀 |
| 목표주가 괴리 | ≥ +20% | `--green` | 배지 |
| 목표주가 괴리 | ≤ -10% | `--red` | 배지 |

### 1.5 배지 색상 (배경/텍스트 페어)

| 종류 | 배경 | 텍스트 |
|---|---|---|
| BUY (📈) | `rgba(29,158,117,0.18)` | `--green` |
| HOLD (⚪) | `rgba(136,135,128,0.18)` | `--gray` |

| 패시브 (📌) | `rgba(58,158,158,0.15)` | `--teal` |

## 2. 타이포그래피 (Typography)

### 2.1 폰트 스택
```
-apple-system, BlinkMacSystemFont, "Segoe UI", "Malgun Gothic", sans-serif
```
- 한국어: `Malgun Gothic` 우선 (Windows), 시스템 한글 폰트 fallback
- 코드/모노: `Consolas, Monaco, monospace`

### 2.2 크기 토큰

| 토큰 | 크기 | 용도 |
|---|---|---|
| `--fs-xs` | 10px | 보조 라벨, 뱃지 |
| `--fs-sm` | 11px | 보조 텍스트, 디버그 패널 |
| `--fs-base` | 13px | 본문 기본 |
| `--fs-md` | 14px | KPI 라벨 |
| `--fs-lg` | 18px | 종목명, 헤더 |
| `--fs-xl` | 22px | KPI 값 |

### 2.3 굵기

- 일반: 400
- 강조: 600 (KPI 값, 헤더, 배지)
- 굵음 강조 (`<strong>`): 700 — 인사이트 패널 핵심 수치

## 5. Shadow

| 토큰 | 값 | 용도 |
|---|---|---|
| `--sh-card` | none | 기본 카드 (다크모드는 그림자 없음) |

## 6. 수치 표기 (Number Formatting)

`3-skills-viz.md` "수치 표기 규칙"의 단일 출처. 표기 일관성을 위해 본 파일에서 정의한다.

| 유형 | 형식 | 예시 |
|---|---|---|
| 수익률 | 부호 + 소수 2자리 + % | `+82.89%`, `-3.21%` |
| 금액 (억원) | 콤마 + "억원" | `1,526억원` |
| 가격 (원) | 콤마 + "원" | `654,500원` |
| 비중 | 소수 1자리 + % | `26.2%` |
| PER / PBR | 소수 2자리 + "x" | `5.71x` |
| Sharpe | 소수 2자리 | `1.23` |
| Beta | 소수 2자리 | `0.88` |
| MDD / VaR | 소수 2자리 + % | `-12.34%` |
| ROE / 변동성 | 소수 1자리 + % | `15.3%` |
| null·없음 | 대시 | `—` (빈칸·0 금지) |
| 배지 괴리율 | 부호 + 소수 1자리 + % | `+24.5%`, `-8.2%` |

## 7. 사용 규칙

- ❌ 다른 Skills 파일에서 hex 코드를 직접 쓰지 않는다 (`#1D9E75` 등). 본 파일을 인용
- ✅ 토큰명만 참조: "양수 → `--green` (`design-tokens.md` §1.2)"
- ✅ 새 색상이 필요하면 본 파일에 토큰을 추가한 뒤 사용
- ✅ 데이터셋 교체 시에도 본 파일은 변경하지 않는다 (디자인 정체성 유지)
